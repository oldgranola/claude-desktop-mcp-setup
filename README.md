# Claude Desktop + Local MCP Servers
## Complete Setup Guide for Linux Mint 22.3 (Debian)

This guide documents a real, tested setup for running Claude Desktop on Linux with seven local MCP servers.

See [Claude-Desktop-MCP-Setup-Guide.md](Claude-Desktop-MCP-Setup-Guide.md) for the full guide.

### What you get

- Claude Desktop for Linux (community Debian port) installed and running
- Seven active MCP servers: filesystem, excel, git, github, fetch, brave-search, podman
- API keys and tokens stored securely, separate from all config files
- A global CLAUDE.md instruction file that shapes Claude's behavior
- Key operational gotchas specific to the Linux port
